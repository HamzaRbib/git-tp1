What is Git, and what is it used for?

Git is a version controle that tracks changes in any set of computer files, it is used for coordinating work among programmers who are collaboratively developing source code during software development.

What is a repository in Git, and what is it used for?

A Git repository is a central storage location for managing and tracking changes in files and directories, it is used to store all the files and source code of a project in github servers so collaborators can get access to the project and contribut to it.

How do you create a new Git repository?

To create a new repository login to your github acount click on the plus button on the top right and click on new repository.

How do you stage changes in Git, and what is the purpose of staging?

After editing your files in the working directory and you whant to add theme to the staging area use the command git add <edited-file>, it is used to help you prepare the changes that will be commited next.

How do you commit changes in Git, and what is the purpose of committing?

You commit changes in git by using the command git commit -m "you comment or message for the commit", the perpose of committing is to keep track of all the changes in your project.

How do you create and switch between branches in Git, and what is the purpose of branching?

You can create and switch between branches using the command git checkout -b "branch name", the purpose of branching is to saperate any changes for being made directly to the main project to maintain stability and make it easy to fix bugs or add new features.

How do you merge branches in Git, and what is the purpose of merging?

You merge branches by using the command git merge "branch name", the purpose of merging is to add any bug fix or feature to the main project branch.

How do you resolve conflicts in Git when merging branches?

The easiest way to resolve a conflicted file is to open it and make any necessary changes.

How do you push changes from your local repository to a remote repository, and what is the purpose of pushing?

You can push changes by using the command git push origin master, the purpose of pushing is to upload you changes to github servres.

How do you fetch and merge changes from a remote repository, and what is the purpose of fetching?

You can fetch changes by using the command git fetch and to merge git merge, fetching is used to get refrence to all changes in the remote repository.
